Reducer


O método reduce executa uma função reducer que é fornecida pelo programador.
Cada array tem elementos nele e com o reducer é possivel unir cada item do array em um unico elemento, resultando num único valor de retorno ao unir cada elemento do array.

Map

O método invoca a função callback passada por argumento para cada elemento do Array e devolve um novo Array como resultado.Não modifica a array antigo realizando as oprações em ordem ou seja a medida qe os elementos aparecem vai realizando operações.
